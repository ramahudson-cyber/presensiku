## Fix: Status pill colors light mode

**Masalah:** `bg-black text-white` — light mode override `text-white` jadi dark (`#0f172a`), tapi `bg-black` tetap hitam. Text gak keliatan.

**Fix:** Ganti status pill pake green-yellow (brand success color) — works both modes:
- Valid: `bg-green-yellow/15 text-green-yellow border border-green-yellow/30` + dot green-yellow
- Invalid: `bg-red-500/10 text-red-400 border border-red-500/20` + dot red
- Checking: `bg-black/5 text-black/50 border border-black/10` + dot gray
